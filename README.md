# java-filmorate — сервис рекомендаций фильмов

Java, Spring Boot, Maven, REST API, SQL, H2, JDBC, JdbcTemplate, Lombok, Git

## О проекте

**Filmorate** — сервис рекомендаций фильмов в формате «Кинопоиск для друзей».  
Пользователи могут искать фильмы, ставить оценки, добавлять друзей, получать рекомендации и видеть активность своего окружения.

Проект в части добавления функциональности разработан **в команде**, при этом **я выступал тимлидом**.

## Основные возможности сервиса

Сервис предоставляет следующие возможности:

- просмотр общих с другом фильмов с сортировкой по популярности;
- вывод топ-N фильмов по количеству лайков;
- поиск фильмов по названию и по режиссёру;
- просмотр оценок пользователей;
- получение рекомендаций фильмов на основе похожих вкусов;
- вывод всех фильмов режиссёра:
  - отсортированных по количеству лайков;
  - отсортированных по годам;
- просмотр ленты событий платформы:
  - добавление и удаление друзей;
  - лайки;
  - отзывы, оставленные друзьями пользователя.

## Реализованный функционал

### Общие фильмы
- вывод общих с другом фильмов;
- сортировка по популярности.

### Удаление фильмов и пользователей
- удаление фильма по идентификатору;
- удаление пользователя по идентификатору.

### Популярные фильмы
- вывод топ-N фильмов по количеству лайков;
- фильтрация:
  - по жанру;
  - по году выпуска.

### Поиск
- поиск по названию фильма;
- поиск по режиссёру.

### Рекомендации
- формирование списка фильмов, которые:
  - понравились пользователям с похожими вкусами;
  - ещё не были просмотрены текущим пользователем.

### Лента событий
- отображение последних событий платформы:
  - добавление в друзья;
  - удаление из друзей;
  - лайки;
  - отзывы.

### Отзывы
- отзывы к фильмам;
- наличие рейтинга и дополнительных характеристик.

### Фильмы по режиссёрам
- добавление информации о режиссёре к фильмам;
- вывод фильмов режиссёра:
  - по количеству лайков;
  - по годам выпуска.
